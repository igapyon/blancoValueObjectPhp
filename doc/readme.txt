blancoValueObjectPhpは 「バリューオブジェクト定義書(php)」というExcel様式を記入するだけで 簡単に バリューオブジェクト クラス・ソースコードが作成できるようにするためのツールです。
PHPのソースコード自動生成に対応しています。
加えて、blancoValueObjectPhpを導入すると、設計書とソースコードが自動的に同期化されるという効果があります。
